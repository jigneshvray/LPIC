
dpkg is a tool to install, build, remove and manage Debian packages

**command**

dpkg options  (.deb package name)

![[dpkg - 1.png]]

![[dpkg - 2.png]]

**Installing Package**

sudo dpkg -i [package name]

**Removing a Package**

sudo dpkg -r [package name]

Updating your repositories

sudo dpkg --update-avail
